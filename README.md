# is Valid URL

Validation for URL input.


## Requires

- [Node.JS](http://nodejs.org/)


## Usage

```javascript
var urlInput = getElementById('contact-url');
isValid.url(urlInput);
```


## Includes

- [Mocha](http://visionmedia.github.io/mocha/)
- [Chai](http://chaijs.com/)
